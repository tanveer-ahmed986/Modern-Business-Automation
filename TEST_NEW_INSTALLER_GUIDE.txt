    STEP 2: COMPLETE UNINSTALLATION
================================================================================

Run the automatic uninstall script:

    D:\gemini_modern_business_automation_system\UNINSTALL_MBAS_FOR_TESTING.bat

Or manual uninstall:

1. Stop MBAS:
   - Press Ctrl+Alt+Delete → Task Manager
   - Find all python.exe and pythonw.exe processes
   - Click "End Task" on each

2. Delete installation folder:
   - Open File Explorer
   - Navigate to: C:\MBAS_TEST
   - Delete the entire folder
   - If Windows says "file in use", restart computer and try again

3. Remove shortcuts:
   - Desktop: Delete MBAS shortcut
   - Start Menu: Remove any MBAS entries

4. Restart your computer (recommended)

================================================================================
    STEP 5: COMMON ISSUES DURING TESTING
================================================================================

Issue: "Installation failed"
Fix:
  - Run installer as Administrator
  - Disable antivirus temporarily
  - Ensure 2GB free disk space

Issue: "Port 8000 already in use"
Fix:
  - Restart computer
  - Make sure old MBAS is fully uninstalled
  - Check no other apps using port 8000

Issue: "Python not found"
Fix:
  - Install Python 3.11 or 3.12 from python.org
  - During Python install, check "Add Python to PATH"
  - Restart installer

Issue: "CMD window appears"
Fix:
  - This shouldn't happen with new installer
  - If it does, note this as a bug
  - Check you're using START_MBAS_TRAY.bat not START_MBAS_DIRECT.bat

Issue: "404 errors for CSS/JS"
Fix:
  - This was fixed in the new installer
  - If it happens, the installer wasn't rebuilt correctly
  - Check frontend/dist/assets/ folder exists

Issue: "Tray icon doesn't appear"
Fix:
  - Wait 30 seconds (first startup takes time)
  - Check Task Manager for pythonw.exe process
  - If running, check system tray (click ^ arrow to show hidden icons)
