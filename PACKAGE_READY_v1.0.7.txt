   - Ask for Windows version

2. COMMON ISSUES

   "Must login every time browser opens"
   → This is INTENTIONAL security feature in v1.0.7!
   → Prevents unauthorized access
   → Working as designed

   "Session expired after 2 hours"
   → This is INTENTIONAL security feature!
   → Enhanced protection against hijacking
   → Simply login again to continue

   "System tray icon not visible"
   → Check Windows taskbar settings
   → Enable "Show hidden icons"
   → Click up arrow (^) in taskbar

   "Background mode: how to stop?"
   → Right-click tray icon → Exit
   → Or use STOP_MBAS.bat

   "Python not found"
   → Install Python 3.11/3.12, check "Add to PATH"

   "Installation fails"
   → Run as Administrator, check internet

   "Port 8000 in use"
   → Right-click tray icon → Exit
   → Or run STOP_MBAS.bat
   → Or restart PC

   "Page won't load"
   → Check tray icon status (green = running)
   → Check backend console for errors
   → Try http://127.0.0.1:8000

3. ESCALATION
   - Collect diagnostic info
   - Remote desktop support
   - Fresh installation if needed
