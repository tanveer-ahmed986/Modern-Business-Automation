STEP 1: UNINSTALL OLD VERSION
------------------------------
Option A (Automatic):
  Double-click: UNINSTALL_MBAS_FOR_TESTING.bat

Option B (Manual):
  1. Press Windows key
  2. Type: "Add or remove programs"
  3. Find: "MBAS - Modern Business Automation System"
  4. Click: Uninstall

Then RESTART your computer (important!)

STEP 2: INSTALL NEW VERSION
----------------------------
1. Navigate to:
   D:\gemini_modern_business_automation_system\MBAS_v2.0_CLIENT_PACKAGE\

2. Double-click:
   MBAS_Professional_Setup_v2.0.1_ZT_Technologies.exe

3. Click "Yes" if Windows asks for administrator permission

4. Follow the installation wizard:
   - Accept license
   - Choose install location (default: C:\MBAS)
   - Select options (desktop icon recommended)
   - Click Install

5. WAIT 3-5 MINUTES (installation shows progress bar now)
   - Creating virtual environment...
   - Upgrading pip...
   - Installing dependencies... (PROGRESS BAR VISIBLE)
   - Initializing database...

6. Click Finish when done

STEP 3: FIRST STARTUP
----------------------
1. Double-click desktop icon: "MBAS"

2. Look for system tray icon (near clock):
   - Icon appears (GRAY color)
   - Wait 30-45 seconds (THIS IS NORMAL for first time!)
   - Icon turns GREEN when ready

3. Browser opens automatically to:
   http://localhost:8000

================================================================================
    TROUBLESHOOTING
================================================================================

Issue: "Installation taking too long"
--------------------------------------
✓ This is NORMAL - installation takes 3-5 minutes
✓ Look for progress bar during "Installing dependencies"
✓ If progress bar is moving, it's working (be patient)
✓ Depends on internet speed for downloading packages

Issue: "First startup taking too long"
---------------------------------------
✓ First startup takes 30-45 seconds (NORMAL)
✓ Look at system tray icon - is it there?
✓ Is it GRAY? Wait... it will turn GREEN
✓ Database initializing in background
✓ Subsequent startups will be 5-10 seconds

Issue: Still getting timeout error
-----------------------------------
✗ Contact: zttechnologies12@gmail.com
✗ Include: Screenshot of error
✗ This should be fixed in v2.0.1

Issue: Installation fails
--------------------------
1. Check Python is installed: python --version
   - Need Python 3.11 or 3.12
   - Download from: python.org

2. Check internet connection
   - Installer downloads packages from PyPI
   - Need stable internet for 3-5 minutes

3. Disable antivirus temporarily
   - Some antivirus block Python installations
   - Re-enable after install completes

4. Run as administrator
   - Right-click installer
   - Select "Run as administrator"
