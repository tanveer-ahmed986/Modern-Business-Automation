================================================================================
    TECHNICAL SUPPORT
================================================================================

If you encounter build issues:

1. Check error messages in build output
2. Verify all prerequisites are installed
3. Review: BUILD_PROFESSIONAL_INSTALLER_README.md (detailed guide)
4. Check: deployment\build_installer\output\Setup Log *.txt (installer logs)

Common issues:

Issue: "Inno Setup not found"
Fix: Install Inno Setup 6 from https://jrsoftware.org/isdl.php

Issue: "Frontend build failed"
Fix:
  cd frontend
  rm -rf node_modules
  npm install
  npm run build

Issue: "npm not found"
Fix: Install Node.js from https://nodejs.org/
