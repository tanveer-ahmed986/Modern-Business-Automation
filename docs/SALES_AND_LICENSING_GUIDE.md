## 6. License Generation

### 6.1 License System Overview

**How It Works:**
1. You (vendor) have a **private key** (NEVER share this)
2. Customer gets a **license file** signed with private key
3. MBAS application verifies license using embedded **public key**
4. License contains:
   - Customer name
   - Package tier (basic/standard/premium)
   - Feature flags
   - Expiry date (for subscriptions)
   - Machine ID hash (optional - for hardware lock)

### 6.2 License Generator Tool Setup

**First-Time Setup:**

**Step 1: Generate Key Pair**
```bash
cd D:\gemini_modern_business_automation_system\tools

# Run license generator for first time
python license_generator.py --generate-keys

# Output:
# ✓ Generated RSA-4096 key pair
# ✓ Private key saved: tools/private_key.pem
# ✓ Public key saved: backend/src/embedded/public_key.pem
```

**⚠️ CRITICAL SECURITY:**
- **NEVER** share `private_key.pem` with anyone
- **NEVER** commit it to GitHub/version control
- **BACKUP** `private_key.pem` to secure location (encrypted USB, password manager)
- If lost, cannot generate compatible licenses
- Store in: Password manager, encrypted drive, bank locker

**Step 2: Verify Setup**
```bash
# Check if keys exist
dir tools\private_key.pem
dir backend\src\embedded\public_key.pem

# Both should exist
```

## 7. Installation Guide for Customers

### 7.1 Installation Steps (Provide to Customer)

**📋 MBAS Installation Guide**

**System Requirements:**
- Windows 10 or 11
- 4 GB RAM minimum
- 5 GB free disk space
- Administrator access

**Step 1: Run Installer**
1. Double-click `MBAS_v1.0_Setup.msi`
2. Click "Next" on welcome screen
3. Accept license agreement
4. Choose installation folder (default recommended)
5. Click "Install"
6. Wait for installation (2-3 minutes)
7. Click "Finish"

**Step 2: Install License**
1. Locate your license file (e.g., `ABC_Pharmacy_Standard_MBAS.license`)
2. Copy the license file
3. Navigate to: `C:\ProgramData\MBAS\`
   - If folder doesn't exist, create it
4. Paste the license file
5. Rename it to: `mbas.license` (exactly this name)

**Step 3: First Launch**
1. Double-click MBAS desktop icon
2. Application starts and verifies license
3. If license valid, you'll see login screen
4. If license invalid, error message appears

**Step 4: First Login**
- Username: `admin`
- Password: `admin123`

**Step 5: Change Password**
1. After login, click your username (bottom left)
2. Click "Change Password"
3. Enter new secure password
4. Save

**Step 6: Configure Business**
1. Go to Settings
2. Update:
   - Business Name: [Your Business Name]
   - Address, Phone, Email
   - Tax Rate
   - Currency
3. Save Settings

**✅ Installation Complete!**

**Troubleshooting:**

❌ **"License not found" error**
→ Ensure license file is at: `C:\ProgramData\MBAS\mbas.license`

❌ **"Invalid license" error**
→ Contact support with error details

❌ **Application won't start**
→ Restart computer and try again

## 8. Subscription Model

### 8.1 Subscription Types

**Monthly Subscription:**
- Payment: Every 30 days
- Auto-renewal: Yes (with payment reminder)
- Cancellation: Anytime, access until end of period
- Grace Period: 7 days after expiry

**Annual Subscription:**
- Payment: Once per year
- Auto-renewal: Optional
- Discount: ~40% vs monthly
- Grace Period: 14 days after expiry

**Perpetual + Maintenance:**
- Initial: One-time purchase
- Maintenance: Annual fee for updates/support
- Maintenance optional after year 1
- Without maintenance: No updates, limited support

## Appendix D: Frequently Asked Questions (Vendor)

**Q: What if customer loses their license file?**
A: Regenerate with same details and send via email. Keep record of all licenses generated.

**Q: Can one license be used on multiple computers?**
A: Perpetual licenses: Technically yes, but license agreement says single installation.
Subscription: Use machine-locking for strict control.

**Q: What if customer wants refund?**
A: Offer 30-day money-back guarantee. Issue prorated refund for subscriptions.

**Q: Can customer downgrade from Premium to Standard?**
A: Yes, at next renewal. No refund for current period. Generate new license for lower tier.

**Q: What if customer modifies license file?**
A: License verification will fail. System won't start. They must contact you for valid license.

**Q: How to handle expired subscriptions?**
A: Grace period of 7 days. After that, system shows "License Expired" error.
Customer can still access to export data but cannot do new transactions.
