================================================================
   🎓 CUSTOMER SUPPORT GUIDE
================================================================

When clients contact you:

1. FIRST RESPONSE
   - Ask for HEALTH_CHECK.bat output
   - Ask for Python version (python --version)
   - Ask for Windows version
   - Ask for error screenshots

2. COMMON ISSUES
   "Python not found"
   → Install Python 3.11/3.12, check "Add to PATH"

   "Installation fails"
   → Run as Administrator, check internet

   "Port 8000 in use"
   → Run STOP_MBAS.bat, or restart PC

   "Page won't load"
   → Check backend console for errors
   → Try http://127.0.0.1:8000

   "Currency not changing"
   → Hard refresh browser (Ctrl+Shift+R)
   → Verify using v1.0.6 package

3. ESCALATION
   - Collect diagnostic info
   - Remote desktop support
   - Fresh installation if needed
