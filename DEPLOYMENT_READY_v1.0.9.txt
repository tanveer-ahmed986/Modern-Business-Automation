================================================================================
  ROLLBACK PROCEDURE
================================================================================

If you need to revert to v1.0.8:

1. Stop v1.0.9:
   Right-click tray icon > Exit
   OR run STOP_MBAS.bat

2. Restore v1.0.8:
   Extract MBAS_v1.0.8_Production_Ready.zip
   Overwrite all files

3. Restore Database:
   Copy backed up mbas_database.db
   Paste to backend/ folder

4. Start v1.0.8:
   Run AUTO_START_WITH_RECOVERY.bat

Note: v1.0.9 makes NO database changes, rollback is 100% safe.
