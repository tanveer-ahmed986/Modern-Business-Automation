TECHNICAL EXPLANATION:
---------------------
Why Decimal vs Float for Money?

Float:
  0.1 + 0.2 = 0.30000000000000004  ❌ Wrong!

Decimal:
  Decimal("0.1") + Decimal("0.2") = 0.3  ✅ Correct!

Fix Applied:
  # Convert float to Decimal via string
  unit_price = Decimal(str(product.selling_price))
  cost_price = Decimal(str(product.cost_price))
  subtotal = unit_price * quantity  # All Decimal now
  total += subtotal  # ✅ Works!
