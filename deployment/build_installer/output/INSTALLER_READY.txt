TROUBLESHOOTING
================================================================================

Issue: Installation hangs or takes too long
Solution: Wait up to 15 minutes (Python package installation)

Issue: "Python not found" error
Solution: Install Python 3.11 or 3.12 from python.org
         Check "Add Python to PATH" during installation

Issue: Permission denied errors
Solution: Install to C:\MBAS instead of Program Files
         OR run installer as Administrator

Issue: Antivirus blocks installation
Solution: Temporarily disable antivirus
         OR add exception for MBAS_Setup_v1.2.1_NoTaskkill.exe

Issue: Installation fails
Solution: Check installation log in %TEMP%\Setup Log*.txt
         Run DIAGNOSE_AND_FIX.bat after partial installation
