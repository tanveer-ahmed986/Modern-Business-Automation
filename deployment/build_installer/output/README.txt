SYSTEM REQUIREMENTS:
--------------------
- Windows 10 or Windows 11 (64-bit)
- 4GB RAM minimum (8GB recommended)
- 2GB free disk space
- Internet connection (for initial setup)
- Python 3.11 or 3.12 (installer will check)

INSTALLATION:
-------------
1. Double-click: MBAS_Professional_Setup_v2.0.0_ZT_Technologies.exe
2. Follow the installation wizard
3. Installation takes 5-10 minutes
4. MBAS will launch automatically after installation

DEFAULT LOGIN:

IMPORTANT: Change this password immediately after first login!

GETTING STARTED:
----------------
1. After installation, find MBAS icon on your desktop
2. Double-click to start MBAS
3. Wait 20-30 seconds for first startup
4. Browser opens automatically to http://localhost:8000
5. Login with credentials above
6. Start managing your business!

TROUBLESHOOTING:
----------------
If MBAS doesn't start after installation:
1. Check that Python 3.11 or 3.12 is installed
2. Look for MBAS icon in system tray (near clock)
3. Check installation log in: C:\Program Files\MBAS\
4. Contact support: zttechnologies12@gmail.com

UNINSTALL:
----------
To uninstall MBAS:
1. Open Control Panel
2. Programs and Features
3. Find "MBAS - Modern Business Automation System"
4. Click Uninstall
5. Follow the prompts

Your data will be backed up before uninstall.
