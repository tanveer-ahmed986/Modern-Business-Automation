STEP 1: Test on Your Current System (Quick Test)
-------------------------------------------------
This verifies the installer works on a system with Python already installed.

1. Close any running MBAS instances:
   - Run: C:\MBAS\STOP_MBAS.bat (if exists)
   - Or: Kill all python.exe processes in Task Manager

2. Run the installer:
   - Double-click: MBAS_Setup_v1.1.0_Simple.exe
   - Click through the wizard
   - Accept default installation path (C:\Program Files\MBAS)
   - Select "Create desktop shortcut"
   - Wait for installation (5-10 minutes for dependencies)

3. Verify installation:
   - Desktop shortcut created: "MBAS - Modern Business Automation System"
   - Start Menu folder created: MBAS
   - Files installed to: C:\Program Files\MBAS

4. Launch MBAS:
   - Double-click desktop shortcut
   - OR: Start Menu > MBAS > MBAS
   - Wait 10-15 seconds for startup

5. Test functionality:
   - Browser opens to: http://localhost:3000
   - Login page appears
   - Login with: admin / admin123
   - Dashboard loads successfully
   - Test creating a product
   - Test all menu items

6. Test uninstaller:
   - Start Menu > MBAS > Uninstall MBAS
   - Verify all files removed
   - Verify desktop shortcut removed

STEP 2: Test on Clean Windows VM (Recommended)
-----------------------------------------------
This simulates a real client installation on a fresh system.

PREREQUISITES:
   - Windows 10 or 11 VM (clean install)
   - NO Python installed initially
   - Internet connection

   2. Run the installer
   3. Installer should warn: "Python 3.11 or 3.12 is required"

B. WITH PYTHON (Should succeed):
   1. Install Python 3.12.1:

      - Run installer
      - ✓ CHECK: "Add Python to PATH"
      - ✓ CHECK: "Install for all users" (optional)
      - Complete installation
      - Restart VM

   2. Verify Python:
      - Open Command Prompt
      - Type: python --version
      - Should show: Python 3.12.1

   3. Run MBAS installer:
      - Double-click: MBAS_Setup_v1.1.0_Simple.exe
      - Follow wizard
      - Wait for dependency installation (5-10 minutes)
      - Choose to launch MBAS when complete

   4. Verify MBAS works:
      - System tray icon appears
      - Browser opens automatically
      - Login page loads
      - Default credentials work (admin / admin123)
      - All features accessible

   5. Test restart:
      - Close MBAS
      - Launch from desktop shortcut
      - Verify it starts correctly

   6. Test uninstaller:
      - Uninstall from Start Menu
      - Verify complete removal

================================================================================
    CLIENT INSTALLATION INSTRUCTIONS
================================================================================

Provide these instructions to clients:

PREREQUISITES (One-time setup):
-------------------------------
1. Install Python 3.12.1:
   - Download from: https://www.python.org/ftp/python/3.12.1/python-3.12.1-amd64.exe
   - Run the installer
   - ✓ IMPORTANT: Check "Add Python to PATH"
   - Click "Install Now"
   - Wait for completion
   - Restart your computer

2. Verify Python installation:
   - Open Command Prompt
   - Type: python --version
   - You should see: Python 3.12.1

MBAS INSTALLATION:
------------------
1. Double-click: MBAS_Setup_v1.1.0_Simple.exe

2. If you see "Windows protected your PC":
   - Click "More info"
   - Click "Run anyway"
   - This is normal for new software

3. Follow the installation wizard:
   - Click "Next" on welcome screen
   - Accept license agreement
   - Choose installation location (default: C:\Program Files\MBAS)
   - Select options:
     ✓ Create desktop shortcut (recommended)
     ☐ Start automatically with Windows (optional)
   - Click "Install"
   - Wait 5-10 minutes (downloads dependencies)

4. When installation completes:
   - Choose "Launch MBAS" and click "Finish"
   - OR: Launch later from desktop shortcut

FIRST STARTUP:
--------------
1. After launching MBAS:
   - Wait 10-15 seconds for system to start
   - System tray icon appears (near clock)
   - Browser opens automatically to login page

2. Login with default credentials:
   - Username: admin
   - Password: admin123

3. IMPORTANT: Change password immediately:
   - Click user menu (top right)
   - Select "Settings"
   - Change password

TROUBLESHOOTING:
----------------
If MBAS doesn't start:
   1. Go to: C:\Program Files\MBAS
   2. Find: EMERGENCY_FIX.bat
   3. Right-click > Run as administrator
   4. Wait 5-10 minutes
   5. MBAS will start automatically

For detailed help:
   - Open: C:\Program Files\MBAS\START_HERE_IF_PROBLEMS.txt

================================================================================
    COMMON ISSUES AND SOLUTIONS
================================================================================

ISSUE: "Python not found" during installation
SOLUTION: Install Python 3.12.1 and restart computer

ISSUE: Installation fails at dependency step
SOLUTION:
  - Check internet connection
  - Run installer as Administrator
  - Disable antivirus temporarily

ISSUE: MBAS doesn't start after installation
SOLUTION:
  - Run: C:\Program Files\MBAS\EMERGENCY_FIX.bat as Administrator
  - Or: Run START_MBAS_TRAY_DEBUG.bat to see error details

ISSUE: "Smart App Control blocked this file"
SOLUTION:
  - Click "More info" > "Run anyway"
  - Or: After installation, run UNBLOCK_FILES.bat

ISSUE: Browser doesn't open automatically
SOLUTION:
  - Manually open browser
  - Go to: http://localhost:3000
  - Or wait 15 seconds and try again

ISSUE: Port 8000 already in use
SOLUTION:
  - Open Command Prompt as Administrator
  - Type: netstat -ano | findstr :8000
  - Find PID (last column)
  - Type: taskkill /F /PID [number]
  - Restart MBAS
