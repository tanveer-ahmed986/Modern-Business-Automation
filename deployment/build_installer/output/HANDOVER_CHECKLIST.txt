COMMON CUSTOMER QUESTIONS
========================================================================

Q: Do I need internet connection?
A: Only for initial setup to download Python dependencies.
   After that, MBAS works completely offline.

Q: Can I install on multiple computers?
A: Depends on your licensing model. Update based on your license terms.

Q: What if I need to move to another computer?
A: 1. Backup database (Settings → Backup)
   2. Install MBAS on new computer
   3. Restore database (Settings → Restore)

Q: How do I update to new version?
A: Run new installer. Your data is automatically preserved.

Q: Is my data safe?
A: Yes. All data stored locally on your computer.
   No cloud services. Regular backups recommended.

Q: What if something goes wrong?
A: Run EMERGENCY_FIX.bat from installation folder
   Or contact support: [Your Contact Info]

========================================================================
TROUBLESHOOTING - QUICK REFERENCE
========================================================================

Issue: Python not found
Fix: Install Python 3.11 or 3.12, check "Add to PATH"

Issue: Port 8000 in use
Fix: Restart computer or close conflicting application

Issue: Slow first startup
Fix: Normal! First run takes 30-60 seconds

Issue: Browser doesn't open
Fix: Manually go to http://localhost:5173

Issue: Can't uninstall (hanging)
Fix: This is FIXED in v1.2.0! New installer won't hang.

Issue: Installation fails
Fix: Run as Administrator, check antivirus settings
