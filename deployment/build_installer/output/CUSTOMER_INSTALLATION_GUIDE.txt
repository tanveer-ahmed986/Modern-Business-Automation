QUICK START (3 Easy Steps)
========================================================================

1. INSTALL PYTHON (If not already installed)
   Download: https://www.python.org/downloads/
   Version: Python 3.11 or 3.12

   IMPORTANT: During Python installation:
   ✓ Check "Add Python to PATH"
   ✓ Click "Install Now"

2. RUN THE INSTALLER
   Double-click: MBAS_Setup_v1.2.0_Robust.exe

   - Click "Next" through the wizard
   - Choose installation path (Recommended: C:\MBAS)
   - Check "Create desktop shortcut"
   - Click "Install"
   - Wait 2-4 minutes

3. LAUNCH MBAS
   - Double-click desktop shortcut: "MBAS - Modern Business Automation System"
   - Wait 30-60 seconds for first startup
   - Browser opens automatically to: http://localhost:5173

   Default Login:
   Username: admin
   Password: password

   IMPORTANT: Change password after first login!

========================================================================
TROUBLESHOOTING
========================================================================

Problem: "Python not found" during installation
Solution: Install Python 3.11 or 3.12
         Make sure "Add Python to PATH" was checked

Problem: Browser doesn't open automatically
Solution: Manually open: http://localhost:5173
         Check tray icon is green

Problem: Can't login
Solution: Use default credentials:
         Username: admin
         Password: password

Problem: Port 8000 already in use
Solution: Close other applications using port 8000
         Or restart your computer

Problem: Slow first startup
Solution: This is normal! First startup takes 30-60 seconds
         Subsequent startups are much faster (5-10 seconds)

Problem: Application won't start
Solution: Run Health Check from Start Menu → MBAS → Health Check

For more help:
  - See: C:\MBAS\README_FIRST.txt
  - See: C:\MBAS\START_HERE_IF_PROBLEMS.txt

UNINSTALLING
========================================================================

If you need to uninstall MBAS:

1. Windows Settings → Apps → Apps & features
2. Find "MBAS - Modern Business Automation System"
3. Click Uninstall
4. Follow the wizard

Your database will be automatically backed up to Desktop before removal.

Alternative:
  - Start Menu → MBAS → Uninstall MBAS

========================================================================
DATA BACKUP
========================================================================

Your business data is stored in:
  C:\MBAS\backend\mbas_database.db

Backup Recommendations:
  - Backup this file daily
  - Keep backups on separate drive or cloud storage
  - Use Settings → Backup/Restore for automatic backups

Automatic Backups:
  MBAS automatically backs up your database:
  - Before uninstalling
  - Before upgrading to new version
  - Backups saved to: Desktop\MBAS_Backup
