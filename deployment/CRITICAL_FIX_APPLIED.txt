================================================================
  WHAT WAS FIXED
================================================================

Technical Details:

Before (BROKEN):
  from src.core.db import get_session
  with get_session() as session:  # ERROR!
      ...

After (FIXED):
  from src.core.db import engine
  from sqlmodel import Session
  with Session(engine) as session:  # WORKS!
      ...

The get_session() function is a generator (uses yield) for
FastAPI dependency injection. It cannot be used as a context
manager. The proper way is to use Session(engine) directly.
