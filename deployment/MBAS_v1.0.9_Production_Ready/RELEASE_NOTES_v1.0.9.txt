For New Installations:

  2. Read INSTALLATION_GUIDE.txt
  3. Run INSTALL_BACKEND_FIXED.bat

  6. Login with: admin / admin123

For Existing Users Upgrading from v1.0.8:
  1. Stop current MBAS (Right-click tray icon > Exit OR STOP_MBAS.bat)
  2. Backup your mbas_database.db file
  3. Extract new version files (overwrite)
  4. Run START_MBAS_TRAY.bat
  5. Your data is preserved automatically

================================================================================
  SYSTEM TRAY FEATURES
================================================================================

Tray Icon States:
  🟢 Green Circle = Server Running
  🔴 Red Circle = Server Stopped
  ⚪ Gray Circle = Server Starting

Tray Menu Options:
  - Open MBAS (default double-click)
    Opens MBAS in your default browser
    Auto-starts server if not running

  - Start Server
    Manually start the backend server
    Shows notification when complete

  - Stop Server
    Gracefully stop the backend
    Closes all connections properly

  - Status
    Shows current server status
    Displays server URL

  - Exit
    Stops server and closes application
    Clean shutdown of all processes

Notifications:
  - "MBAS server started successfully!"
  - "MBAS server stopped"
  - "Server Status: Running"
  - Error messages if startup fails
