ISSUE 1: "Smart App Control blocked a file that may be unsafe"
--------------------------------------------------------------
This happens because Windows blocks files downloaded from the internet.

SOLUTION (Option A - Recommended):
1. Right-click UNBLOCK_FILES.bat
2. Select "Run as administrator"
3. Wait for completion
4. Continue with installation

SOLUTION (Option B - Manual):
1. Open Windows Security
2. Go to: App and browser control > Smart App Control settings
3. Turn Smart App Control to "Off"
4. Extract MBAS again to a new folder
5. Continue with installation


ISSUE 2: "Failed to start: Server failed to start" or CMD window blinks
------------------------------------------------------------------------
This indicates MBAS can't start the backend server.

SOLUTION (Step-by-step troubleshooting):

Step 1: Run diagnostic startup
-------------------------------
1. Double-click START_MBAS_TRAY_DEBUG.bat
2. Wait for checks to complete
3. A log file will open showing the exact error
4. Share the log file if you need support


Step 2: Verify Python installation
----------------------------------
1. Open Command Prompt
2. Type: python --version
3. You should see: Python 3.11.x or 3.12.x
4. If not found or wrong version:
   - Download Python 3.12.1 from: https://www.python.org/downloads/
   - Run installer
   - CHECK "Add Python to PATH"
   - Restart computer
   - Run INSTALL.bat again


Step 3: Clean installation
---------------------------
1. Delete the venv folder (if exists)
2. Right-click INSTALL.bat > Run as administrator
3. Wait for installation (2-5 minutes)
4. When complete, run START_MBAS_TRAY_DEBUG.bat


Step 4: Check port availability
-------------------------------
1. Open Command Prompt as Administrator
2. Type: netstat -ano | findstr :8000
3. If you see results, port 8000 is in use:
   - Find the PID number (last column)
   - Type: taskkill /F /PID [number]
4. Run START_MBAS_TRAY_DEBUG.bat again


Step 5: Disable antivirus temporarily
--------------------------------------
1. Temporarily disable Windows Defender/Antivirus
2. Run INSTALL.bat as Administrator
3. Run START_MBAS_TRAY_DEBUG.bat
4. Add MBAS folder to antivirus exclusions
5. Re-enable antivirus


COMPLETE REINSTALLATION (if nothing else works)
================================================

1. STOP MBAS:
   - Double-click STOP_MBAS.bat
   - Open Task Manager (Ctrl+Shift+Esc)
   - End all "python.exe" processes

2. DELETE OLD INSTALLATION:
   - Delete the entire MBAS folder
   - Empty Recycle Bin

3. DISABLE SMART APP CONTROL:
   - Windows Security > App and browser control
   - Smart App Control > Turn Off

4. DISABLE ANTIVIRUS TEMPORARILY:
   - Windows Security > Virus & threat protection
   - Manage settings > Turn off Real-time protection

5. FRESH EXTRACTION:
   - Extract MBAS_v1.0.9_Production_Ready.zip to C:\MBAS
   - DO NOT extract to Desktop, Downloads, or network drives

6. UNBLOCK FILES:
   - Right-click C:\MBAS\UNBLOCK_FILES.bat
   - Run as administrator
   - Wait for completion

7. INSTALL:
   - Right-click C:\MBAS\INSTALL.bat
   - Run as administrator
   - Wait for completion (2-5 minutes)
   - Do NOT close the window until you see "Installation complete"

8. ADD EXCLUSIONS:
   - Windows Security > Virus & threat protection
   - Manage settings > Exclusions > Add exclusion
   - Add folder: C:\MBAS

9. RE-ENABLE ANTIVIRUS:
   - Windows Security > Virus & threat protection
   - Manage settings > Turn on Real-time protection

10. START MBAS:
    - Double-click C:\MBAS\START_MBAS_TRAY.bat
    - Wait 10-15 seconds for startup
    - Browser should open automatically
