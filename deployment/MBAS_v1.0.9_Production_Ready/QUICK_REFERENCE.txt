KEY IMPROVEMENTS:
-----------------
1. All tray dependencies (pystray, Pillow, psutil, requests)
   now installed DURING setup, not on first run

2. Database uses StaticPool (1 connection) instead of
   QueuePool (30 connections), preventing lock contention

3. SQLite optimizations added:
   - WAL mode (concurrent reads)
   - 30-second busy timeout
   - 64MB cache
   - Memory temp storage

4. Graceful watchdog fallback:
   - Auto-recovery if available
   - Direct start if not
   - Always reliable
