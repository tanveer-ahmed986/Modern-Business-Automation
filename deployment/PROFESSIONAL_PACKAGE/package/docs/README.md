## 🚀 Quick Start Guide

### Installation

1. **Download** MBAS installer from www.zttechnologies.org
2. **Run** `MBAS_Professional_Setup_v2.0.0_ZT_Technologies.exe`
3. **Follow** the installation wizard
4. **Wait** for installation to complete (2-5 minutes)
5. **Launch** MBAS from desktop icon or Start menu

### First Launch

1. MBAS will start in **system tray mode** (look for icon near clock)
2. Wait **30 seconds** for initial startup
3. Browser will open automatically to http://localhost:8000
4. **Login** with default credentials:
   - Username: `admin`
   - Password: `admin123`
5. **IMPORTANT:** Change password immediately after first login!

## 💾 Backup and Data Security

### Automatic Backups
MBAS automatically backs up your data daily. Backups are stored in:
```
C:\Program Files\MBAS\backups\
```

### Manual Backup
1. Go to **Settings** → **System Settings**
2. Click **Backup** tab
3. Click **Backup Now**
4. Save backup file to safe location

### Restore from Backup
1. Go to **Settings** → **System Settings**
2. Click **Backup** tab
3. Click **Restore**
4. Select backup file
5. Confirm restoration

**WARNING:** Restoring a backup will replace current data!

## 🆘 Troubleshooting

### MBAS Won't Start
1. Check system tray for MBAS icon
2. If not visible, run `MBAS.exe` from installation folder
3. Check logs in: `C:\Program Files\MBAS\mbas_tray.log`
4. Restart computer and try again

### Can't Access Login Page
1. Wait 30 seconds after startup (first-time startup takes longer)
2. Manually open: http://localhost:8000
3. Check firewall isn't blocking port 8000
4. Check antivirus hasn't quarantined files

### Forgot Password
1. Contact your system administrator
2. Admin can reset password from User Management
3. If you're the admin, contact Z&T Technologies support

### Application Errors
1. Check error logs in installation folder
2. Update to latest version
3. Contact support with error details
