TROUBLESHOOTING:
---------------
If installation fails:
  1. Verify Python 3.11 or 3.12 installed
  2. Check "Add Python to PATH" was selected
  3. Run INSTALL.bat as Administrator
  4. Check internet connection
  5. Run HEALTH_CHECK.bat to diagnose

If you have Python 3.13:
  1. Download Python 3.12.4 from python.org
  2. Install with "Add to PATH" checked
  3. Open new command prompt
  4. Run: python --version (should show 3.12.x)
  5. Run INSTALL.bat again
